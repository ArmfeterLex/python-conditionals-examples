# Python Conditionals Examples

Коллекция простых примеров на Python для изучения условных операторов (if/elif/else) и циклов (for/range).

## Примеры

### 1️⃣ Счетчик сообщений (простой)
Цикл от 0 до 5. Если сообщений > 0 — показывает количество, иначе — "У вас нет сообщений".

### 2️⃣ Утро до 12 часов
Цикл по часам (0-23). До 12 часов выводит "Доброе утро!".

### 3️⃣ Утро/день до/после 12
Цикл по часам. До 12 — утро, после — день.

### 4️⃣ Шкала Бофорта (сила ветра)
Цикл от 0 до 7 с описанием силы ветра: от штиля до шторма.

### 5️⃣ Счетчик сообщений (продвинутый)
Цикл от 0 до 20. Правильные окончания для русского языка:
- 0: "нет новых сообщений"
- 1: "1 новое сообщение"
- 2-4: "X новых сообщения"
- 5-20: "X новых сообщений"

### 6️⃣ Приветствия по времени суток (версия 1)
Цикл по часам с приветствиями:
- 0-5: ночь
- 6-11: утро
- 12-17: день
- 18-23: вечер

### 7️⃣ Приветствия по времени суток (версия 2)
Та же логика, но с явными условиями через and/or.

### 8️⃣ Стих про хорошего/плохого мальчика
Демонстрация if/else с булевой переменной.

### 9️⃣ Завтрак из продуктов
Логика выбора блюда в зависимости от наличия продуктов:
- Есть яйца + молоко → омлет
- Есть яйца без молока → яичница
- Нет яиц, но есть молоко и хлопья → хлопья с молоком
- Только молоко → стакан молока
- Только хлопья → сухие хлопья
- Ничего нет → разгрузочный день

Три варианта с разными комбинациями продуктов.
